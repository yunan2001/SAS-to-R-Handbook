Structure of a shinydashboard App

This guide explains how to structure a basic shinydashboard application in R.

Basic Template

library(shiny)
library(shinydashboard)

dashboardPage(
  dashboardHeader(title = "Basic Dashboard"),
  dashboardSidebar(
    sidebarMenu(
      menuItem("Dashboard", tabName = "dashboard", icon = icon("dashboard")),
      menuItem("Widgets", tabName = "widgets", icon = icon("th"))
    )
  ),
  dashboardBody(
    tabItems(
      tabItem(tabName = "dashboard",
              h2("Dashboard content")
      ),
      tabItem(tabName = "widgets",
              h2("Widgets content")
      )
    )
  )
)

Key Components

  • dashboardHeader(): Contains the title and top-right dropdown menus
  • dashboardSidebar(): Contains the navigation menu
  • dashboardBody(): Displays the content of each tab

Navigation

Use tabItems and tabItem to specify what should be displayed for each menu item.
